# ✦ Todo App

A sleek, dark-themed todo app built with React, featuring glass morphism UI and unique interactive animations.

## Features

- Add and delete tasks
- Click a task to mark it as done with a **realistic glass crack animation** that radiates from your click point
- Delete tasks with a **warp animation** that sucks them out of existence
- **Snow particle background** using particles.js
- Tasks persist across page refreshes using localStorage
- Each crack is completely unique — randomised using the same algorithm as the [CodePen glass crack effect](https://codepen.io/marcellegane/pen/kBYwoO)

## Tech Stack

- React
- HTML5 Canvas API
- particles.js
- html2canvas
- CSS3 animations

## Getting Started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
